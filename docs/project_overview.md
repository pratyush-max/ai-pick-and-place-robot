Project Overview
Project Title

AI-Based Robotic Arm for Object Detection and Pick-and-Place

Introduction

This project aims to develop an intelligent robotic arm capable of detecting objects using computer vision and automatically performing pick-and-place operations. The system combines artificial intelligence, embedded systems, and robotics to create a simple automation solution.

A webcam captures real-time video of the workspace, and the images are processed using computer vision techniques. The object detection model identifies objects in the scene, and the system sends commands to the robotic arm to move and pick the detected object.

Technologies Used

Python

OpenCV for image processing

YOLOv8 for object detection

Arduino Uno R3 for robotic arm control

Servo motors for movement of the robotic arm

3D printed robotic arm components

System Workflow

Webcam captures real-time video feed.

Computer vision system processes the video using OpenCV.

YOLOv8 detects objects in the frame.

Python program sends commands to Arduino.

Arduino controls servo motors to move the robotic arm.

Robotic arm performs pick-and-place operation.

Current Progress

Hardware components selected and purchased

Robotic arm CAD design completed

Object detection code implemented

Arduino servo control code developed

3D printing and system integration planned in the next phase

Expected Outcome

The final system will be able to detect objects using computer vision and automatically move the robotic arm to pick and place them in a desired location.
